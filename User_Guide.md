To run the viewer, go to

    /Users/kendrick/External_Software/tsl/tse
    
Then type
    
    bin/src/tse


Controls:
 - S zooms out
 - W zooms in
 - A goes left
 - D goes right
 - C goes down
 - SPACE goes up
 - Right-click modifies the view
 - SHIFT + \{COMMAND\} makes things go faster

Finally, the repository is located at https://github.com/jovobe/tsl
